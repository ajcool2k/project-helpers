# Raspberry PI

**configuration**

```bash
sudo raspi-config
```

**manage local wifi network**

```bash
sudo nano /etc/wpa_supplicant/wpa_supplicant.conf
```

**blacklist drivers (e.g. wifi / bluetooth)**

```bash
sudo nano /etc/modprobe.d/raspi-blacklist.conf
```
